1. Перед запуском программы установите пути к файлам логов и результатов в конфигурационном файле settings.json в той же директории, что и .exe файл
2. Параметры фильтрации можно передавать либо во время выполнения программы через консоль, либо при вызове программы в качестве ее аргументов в виде: order-filtering <район доставки> <дата доставки>, например order-filtering Ленинский "2024-10-24 16:00:00".
